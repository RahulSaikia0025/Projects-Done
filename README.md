# 🚀 Machine Learning Projects Portfolio

Welcome to my collection of Machine Learning and Computer Vision projects.  
This repository showcases practical implementations of real-world AI applications using Python and deep learning techniques.

---

## 📌 Projects Included

### 🔢 Automatic Number Plate Recognition (ANPR)
A computer vision system that detects and extracts vehicle license plate numbers from images or video streams.

**Features:**
- Vehicle detection
- License plate localization
- OCR-based text extraction
- Real-time processing support

**Tech Stack:**
- Python
- OpenCV
- TensorFlow / PyTorch
- EasyOCR / Tesseract

---

### ✍️ Handwritten Digit Recognizer
A deep learning model that identifies handwritten digits (0–9) using image input.

**Features:**
- Trained on MNIST dataset
- High accuracy digit classification
- Real-time prediction support

**Tech Stack:**
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

---

### 🤟 Sign Language Recognition with Machine Learning
A system that interprets hand gestures into meaningful text or commands.

**Features:**
- Hand tracking and gesture detection
- Real-time sign recognition
- Supports multiple gestures

**Tech Stack:**
- Python
- OpenCV
- MediaPipe
- TensorFlow / PyTorch
