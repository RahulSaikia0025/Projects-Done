# 🔢 Automatic Number Plate Recognition (ANPR)

## 📌 Overview
This project detects and extracts vehicle license plate numbers using computer vision and OCR.

## 🚀 Features
- Vehicle detection
- License plate localization
- OCR text extraction
- Real-time support

## 🛠 Tech Stack
- Python
- OpenCV
- TensorFlow / PyTorch
- EasyOCR / Tesseract

## ▶️ Usage
```bash
python main.py
```
